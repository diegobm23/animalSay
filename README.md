# animalSay
It's a clone of the cowsay app made in Java. Just for fun.

## Running

You'll need to pass two arguments:

```bash
animalSay [animal] [message]
```

Example: 
```bash
java -jar animalSay.jar "cow" "Hello GitHub!"
```
